  ```js
  import { requestId } from 'hono/request-id';
  import { pinoHttp } from 'pino-http';
  app.use(requestId());
  app.use(async (c, next) => {
    c.env.incoming.id = c.var.requestId;
    await new Promise((resolve) => pinoHttp()(c.env.incoming, c.env.outgoing, () => resolve()));
    c.set('logger', c.env.incoming.log);
    await next();
  });
  // handlers: c.var.logger.info(...)
  ```
  Runtime 相依：需要 `@hono/node-server`（已確認 `src/server.ts` 使用）。Item 1a 追加安裝 `pino-http`，Item 4 重寫採用此模式，Item 11 從「加 log 呼叫」改為「從 c.var.logger 發 request-scoped 日誌」。第一輪 stash 的 SubAgent 產出已 drop（與官方範例不一致，重做比 patch 乾淨）。
